SHORT DESCRIPTION
    Provides background information on the Orchestrator Webservice
    
LONG DESCRIPTION
    The Orchestrator web service is a restful webservice. This PowerShell
    module is built to interact with it. The module is built on top of
    Orchestrator.Interop.dll
